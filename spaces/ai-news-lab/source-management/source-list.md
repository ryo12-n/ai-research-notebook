## ステータス定義

| ステータス | 説明 | 遷移条件 |
|-----------|------|---------|
| `candidate` | 新規発見。まだ評価していない | discovery-log に記録後、source-list に追加した時点で付与 |
| `trial` | 試用中。2週間〜1ヶ月の試用期間 | candidate から手動で変更。評価テンプレートで初回評価を実施 |
| `active` | 定常的に参照するソース | trial 期間の評価で総合スコア 3.0 以上かつ継続利用を決定 |
| `inactive` | 一時的に参照を停止 | active から変更。理由を備考に記載。四半期レビューで再評価 |
| `removed` | 完全に除外 | 総合スコア 2.0 未満が継続、またはソース自体が消滅。理由を備考に記載 |

### ステータス遷移図

```
candidate → trial → active
                  ↘ inactive ← active
                  ↘ removed  ← inactive
                               ← trial
```

## カテゴリ一覧

| カテゴリ | 説明 | 例 |
|---------|------|-----|
| ニュースサイト | AI/テック専門のニュースメディア | The Verge AI, TechCrunch AI, VentureBeat |
| 個人ブログ・ニュースレター | 個人が運営するブログやニュースレター | Simon Willison's Weblog, The Batch |

| YouTube・Podcast | 動画・音声メディア | Two Minute Papers |
| 論文リポジトリ | 学術論文のプレプリント・リポジトリ | arXiv cs.AI, Semantic Scholar |
| 企業公式ブログ | AI関連企業の公式テックブログ | OpenAI Blog, Google AI Blog |
| コミュニティ・フォーラム | ディスカッションフォーラム・コミュニティ | Hacker News, Reddit r/MachineLearning |

## 粒度の定義

| 粒度 | 説明 |
|------|------|
| サイト | Webサイト全体を情報源として登録 |
| 人 | 特定の個人（ブロガー、研究者等）を情報源として登録 |
| チャンネル | YouTube チャンネル、Podcast 番組等 |
| ニュースレター | メール配信型の定期刊行物 |

## 運用ルール

- ID は `S-001` から連番で付与する。欠番は再利用しない
- ソースの追加は `discovery-log/` に記録した上で本リストに反映する
- ステータス変更時は `備考` 列に変更理由と日付を追記する
- `最終評価日` は `evaluation-log/` でのスコアリング実施日を記入する
